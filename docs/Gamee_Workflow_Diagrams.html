Gamee Friend Analyzer ワークフロー図（縦型）

遊んだ記録追加

flowchart TD A([開始]) --> B[入力フォーム表示] --> C[ゲーム選択] --> D[日付・時間入力] --> E{入力内容は正常か？} E -- No --> F[エラーメッセージ表示] F --> C E -- Yes --> G[DB登録] --> H[登録完了メッセージ] --> I([終了])

フレンド一覧

flowchart TD A([開始]) --> B[フレンド一覧取得] --> C[一覧表示] --> D{検索条件入力あり？} D -- Yes --> E[条件で絞り込み] E --> F[結果表示] D -- No --> F F --> G{フレンド選択？} G -- Yes --> H[詳細画面へ] G -- No --> I([終了])

フレンド詳細

flowchart TD A([開始]) --> B[フレンド選択] --> C[詳細情報取得] --> D[プレイ履歴取得] --> E[TOP3フレンド集計] --> F[詳細画面表示] --> G{関係図を表示する？} G -- Yes --> H[関係図生成] --> I[関係図表示] --> J([終了]) G -- No --> J

ダッシュボード

flowchart TD A([開始]) --> B[ログイン] --> C[ランキング集計] --> D[最近遊んだフレンド取得] --> E[未プレイフレンド取得] --> F[ダッシュボード表示] --> G{詳細を確認する？} G -- Yes --> H[詳細画面へ] G -- No --> I([終了])

ランキング表示

flowchart TD A([開始]) --> B[プレイ履歴取得] --> C[回数集計] --> D[ランキング生成] --> E[ランキング表示] --> F{期間フィルタ変更？} F -- Yes --> B F -- No --> G([終了])

関係図表示

flowchart TD A([開始]) --> B[プレイ履歴取得] --> C[上位フレンド抽出] --> D[ノード生成] --> E[リンク生成] --> F[ネットワーク描画] --> G{再描画する？} G -- Yes --> B G -- No --> H([終了])
